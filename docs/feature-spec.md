# Feature Specification

## 개인 대시보드

- 오늘 마감
- 이번 주 마감
- 내 미완료 업무
- 지연 업무
- 확인 요청
- 최근 멘션

전체 업무를 한 번에 불러오는 방식은 피하고, 사용자에게 필요한 요약부터 표시합니다.

## 프로젝트 대시보드

- 진행률
- 부서별 완료율
- 미완료/지연 업무
- 최근 댓글
- 첨부
- 주요 메일
- CLAIM/리스크

프로젝트 화면은 리스트, 칸반, 간트/타임라인, 캘린더, 파일, 활동 로그 탭으로 구성합니다.

## 업무 관리

초기 업무 유형:

- 작요
- 생요
- 설검
- 업무참조
- 자재증
- 검사완료
- 출고완료
- 계산서
- CLAIM
- SPARE
- 일반업무

초기 상태:

- 미착수
- 진행중
- 검토요청
- 완료요청
- 완료확인
- 반려
- 보류

## 칸반

기본 컬럼:

- 접수
- 검토중
- 설계중
- 구매/자재
- 품질/검사
- 출고/완료
- 보류

카드에는 업무명, 고객/호선, 담당자, 납기, 지연 여부, 댓글/첨부 수를 표시합니다.

## 통합검색

검색 대상:

- 프로젝트명
- 업무명
- 본문
- 댓글
- 메일 제목/본문
- 첨부파일명
- 고객명
- 회사명
- 호선
- 품목
- 담당자

1차 구현은 PostgreSQL Full Text Search와 trigram 인덱스로 시작합니다. 첨부 OCR과 대규모 검색은 별도 단계에서 OpenSearch를 검토합니다.

## 외부 연동

- Google OAuth: 기본 로그인
- Gmail: 메일 업무 첨부, 고객/호선/품목/납기 후보 추출
- Gemini: 댓글/메일 요약, 유사 이력 검색, 주간보고 초안
- Slack/Telegram/Kakao: 알림과 간단한 업무 조회/상태 변경
- API/CLI/MCP: 모든 핵심 기능을 내부 API로 호출 가능하게 유지
